Installation

You can install Droid in a couple of ways:

1. Standalone PHAR file (recommended)

wget http://droid-php.com/assets/droid.phar -O /usr/local/bin/droid
chmod 755 /usr/local/bin/droid

2. Install as a project dependency

If you plan to use Droid as a task-runner in one of your projects, you can use composer to install Droid as a dependency.

Simply add the following line to your composer.json

"require": {
    "droid/droid": "~1.0",
    "droid/droid-plugins": "~1.0"
}

This will install the Droid application, and a set of default plugins.

Then update your composer dependencies by running:

composer update

You can now access Droid by running:

vendor/bin/droid

Droid is open-source

You can use Droid in your projects and for your server configuration management for free. You can easily extend it's functionality through plugins, and use plugins of others.

For further details, check out github.com/droid-php
