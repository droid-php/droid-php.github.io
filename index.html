Droid

The elegant task runner for devs, ops and other lazy people

Droid will help you automate your builds, tests, deploys, background tasks, and even handle your configuration management

What is Droid?

Droid is a task-runner, similar to Gulp, Grunt, Apache Ant, Phing, etc.

It is built using standard Symfony Console Commands. This means you can use any existing commands to script your builds, tests and deploys.

Furthermore, Droid will just as happily run commands on remote hosts, or host groups, as it performs it's tasks locally. This allows you to use Droid as a Configuration Management tool, much like Ansible, Puppet or Chef. But much simpler, and easier to extend.

Getting started

Droid is open-source

You can use Droid in your projects and for your server configuration management for free. You can easily extend it's functionality through plugins, and use plugins of others.

For further details, check out github.com/droid-php
